# Westlands Mining & Exploration Website

Single-page company website for Westlands Mining & Exploration.

## Structure

- `index.html` contains all HTML, CSS, and JavaScript.
- Images and logo assets live in the same folder as `index.html`.
- The folder is ready for GitHub Pages: publish from the repository root.

## Local Preview

```bash
python3 -m http.server 8766
```

Then open `http://127.0.0.1:8766/index.html`.

## Notes

Replace the contact email and phone details in `index.html` with the company's official contact information before public launch.
